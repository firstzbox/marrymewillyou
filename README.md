# ❤️ Dear Bubu

A proposal website built with Next.js, Tailwind CSS and Framer Motion.

## Run locally

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Open

http://localhost:3000

---

## Build for production

```bash
npm run build
```

Run production

```bash
npm start
```
